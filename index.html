Sci-Fi & Fantasy Girlz

Introduction

The Sci-Fi & Fantasy Girlz is a Goodreads group created on April 5th 2014. The group primarily focuses on science fiction and fantasy written by women. We also discuss works written by men that feature female protagonists/main characters.

I wanted to explore our group. Who are members are, our reading preferences, so I gathered some data via the Goodreads API and analized it with R. On this page is a summary of what I found.

The Books We Have Read

To the date I gathered this data, our group had read 24 books from different categories like epic fantasy, science fiction, historical fiction and even chick lit! (according to Goodreads, anyway)

We started doing two bi-monthly selections, then this year we decided to try something different and started selecting one book each month.

I wanted to compare our group members ratings against Goodreads average ratings. So in the following chart, you will see the group's average rating displayed as green bars, and Goodread's average rating represented by the blue lines.

Group Members

On the date the data was gathered, the Sci-Fi & Fantasy Girlz group had 246 members from all over the world!

Our gender distribution is 61% women, 13% men and 26% not disclosed.

What our members read

I also explored what other kinds of books the members of the group reads. For this, I downloaded a list of all the books read by users and got the shelves for each book on Goodreads. We read many different genres, but we seem to share the love for fantasy and science fiction.

About this analysis

The data gathering and processing required for this analysis were done in R using Goodread's publicly available API. It is based on data captured on April, 2016. After processing the final data was exported in json format in order to be included in this page.

This document was created using Kendo UI for the charts and map. Javascript was used to glue everything together.

I will eventually get my scripts ready for public consumption and share them on Github, but for now, no one should be exposed to that mess :)

Data Privacy

This data was gathered using Goodread's API which respects the privacy setting for users. Data from users with a private profile is not available, so their data is not included in this analysis.
